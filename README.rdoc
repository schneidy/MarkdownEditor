How to use
===

1. Create a new user.
2. Create a markdown paper
3. Type in the markdown box
4. Click "Render" to see how the page looks.
5. Click "Save" to save the markdown or just click on the home button in the navigation bar to go back to your papers.


Uses
===
* [Tilt](https://github.com/rtomayko/tilt)
* [Rdiscount](http://dafoster.net/projects/rdiscount/)
* [Devise](https://github.com/plataformatec/devise)
* [Mongoid](http://mongoid.org/en/mongoid/index.html)
